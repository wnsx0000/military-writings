# 1. 방정식

### 1) 삼차방정식 근과 계수의 관계

> $ax^3 + bx^2 + cx + d = 0 (a \ne 0)$의 세 근이 $\alpha, \beta, \gamma$ 인 경우.<br>
> $\alpha + \beta + \gamma = -\frac{b}{a}$<br>
> $\alpha\beta + \beta\gamma + \gamma\alpha = \frac{c}{a}$<br>
> $\alpha\beta\gamma = -\frac{d}{a}$<br>
